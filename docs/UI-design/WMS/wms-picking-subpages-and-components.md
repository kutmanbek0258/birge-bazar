# WMS — Подстраницы и компоненты страницы «Сборка/Комплектация» (Picking)

---

## Основные подстраницы (разделы) внутри picking:

1. **Список заданий на сборку (Picking Tasks List)**
   - Таблица заданий: номер, статус (новое, в работе, собрано, завершено), маршрут, количество SKU.
   - Поиск и фильтры по статусу, дате, исполнителю.

2. **Детали задания (Picking Task Detail)**
   - Карточка задания: маршрут сборки, список товаров, исполнители.
   - Статусы по каждому товару (ожидает, собрано, не найдено, брак).
   - Время начала/завершения.
   - Действия: “Взять в работу”, “Завершить”, “Проблема”.

3. **Сканирование/комплектация (Picking & Scanning)**
   - Модуль сканирования (BarcodeScanner) для подтверждения отбора.
   - Список отсканированных/неотсканированных товаров.
   - Ручной ввод в случае отсутствия сканера.
   - Кнопки: “Добавить”, “Собрано”, “Брак”.

---

## Примерные URL:

- `/picking` — список заданий
- `/picking/:id` — детали задания
- `/picking/:id/scan` — сканирование и сборка

---

## Основные компоненты:

- **PickingTasksList** — таблица заданий (номер, статус, маршрут, количество, исполнитель, действия)
- **PickingFilters** — фильтры по статусу, дате, исполнителю
- **PickingTaskCard** — карточка задания: маршрут, товары, исполнители, таймер
- **PickingProductList** — список товаров в задании
- **PickingProductRow** — строка товара (SKU, название, статус, количество)
- **PickingStatusBadge** — отображение статуса задания/товара (новое, в работе, собрано, брак)
- **BarcodeScanner** — сканирование для подтверждения сборки
- **PickingManualInput** — ручной ввод (если нет сканера)
- **PickingActions** — кнопки “Взять”, “Собрано”, “Проблема”, “Завершить”
